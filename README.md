## 使用技術
- フロントエンド：Next.js 15.3
- バックエンド：Next.js 15.3、Python 3.13.3（FastAPI0.115.12）
- データベース：Supabase
- デプロイ：Vercel
- バージョン管理：Git、GitHub

※箇条書きは「-」のあとに空白を入れて本文を始めることで可能です。
